よくある質問

入居や施設についてのご質問にお答えします

入居の条件はありますか？

+

原則として、60歳以上で自立または軽度の介護が必要な方を対象としています。詳しい入居条件については、お気軽にお問い合わせください。見学時に詳しくご説明いたします。

介護が必要になった場合はどうなりますか？

+

住宅型有料老人ホームのため、介護が必要な場合は外部の介護サービス事業者と個別にご契約いただきます。事業者選びのご相談や、ケアマネージャーとの連携など、私たちが全面的にサポートいたしますので、ご安心ください。

見学はできますか？

+

はい、随時見学を受け付けております。事前にご予約をお願いいたします。施設内をご案内し、居室や共用スペース、食事の試食（ご希望の場合）も可能です。お気軽にお問い合わせください。

入居時に必要な費用を教えてください

+

入居一時金300万円（返還金制度あり）と、月額費用16万円（管理費、食費、光熱費込み）が基本費用となります。詳細な料金体系については、見学時に詳しくご説明いたします。

食事の対応について教えてください

+

栄養士が監修した食事を1日3食提供しています。常食、刻み食、ペースト食など、お身体の状態に合わせた食事形態に対応可能です。アレルギーや持病による食事制限がある方にも個別対応いたします。

外出や外泊は自由にできますか？

+

はい、基本的に外出・外泊は自由です。お一人での外出が難しい場合は、スタッフがサポートいたします。ご家族との外出・外泊も可能ですので、ご自由にお過ごしください。

ペットを連れて入居できますか？

+

申し訳ございませんが、衛生管理およびアレルギーをお持ちの方への配慮から、ペットとのご入居はお断りしております。ご理解いただけますと幸いです。

医療的なケアは受けられますか？

+

看護師が常駐し、日常的な健康管理を行っています。協力医療機関との連携により、定期的な往診や緊急時の対応も可能です。ただし、常時医療的ケアが必要な場合は、ご相談の上で対応可否を判断させていただきます。

面会時間に制限はありますか？

+

基本的に面会時間は9:00〜18:00となっております。ただし、ご事情がある場合はご相談ください。できる限り柔軟に対応させていただきます。

居室に持ち込める家具や家電はありますか？

+

テレビ、ラジオ、小型冷蔵庫、加湿器などはお持ち込みいただけます。使い慣れた家具もスペースが許す範囲で配置可能です。火気を使用する器具や安全上問題のある設備については制限がありますので、事前にご相談ください。

レクリエーションへの参加は必須ですか？

+

いいえ、レクリエーションへの参加は任意です。ご自身の興味や体調に合わせて自由に参加していただけます。無理に参加していただく必要はございません。

入居までの流れを教えてください

+

①見学・相談 → ②入居申込 → ③健康診断・面談 → ④契約 → ⑤入居準備 → ⑥ご入居、という流れになります。お申込から入居まで通常2〜4週間程度です。詳しくはお問い合わせください。

体験入居はできますか？

+

はい、1泊2日からの体験入居が可能です（有料）。実際の生活を体験していただくことで、入居後のイメージがより明確になります。ご希望の方はお問い合わせください。

退去時の返還金について教えてください

+

入居一時金は5年間で償却いたします。償却期間内に退去された場合、未償却分を返還いたします。詳しい計算方法については、契約時に詳しくご説明いたします。
